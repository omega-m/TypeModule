# Name TypeModule
Unityでタイピングゲームを開発する際に必要となる、共通処理を担当するモジュールです。

このモジュールを使う事で、以下の機能を素早く実装することができます。
・TextBoxを使わずに、キーボードの入力から日本語(英語)の文字列を生成及び取得。
・キーボードの入力から、指定された文字列が正しく打ててるかを確認。(ありがちな、お題の文字を真似して打たせるタイピングゲーム形式のシステム)
